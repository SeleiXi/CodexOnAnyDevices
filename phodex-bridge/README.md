# CodexOnAnyDevices

Local bridge for controlling a Codex runtime from phones, browsers, and other devices.

## Install

```sh
npm install -g codex-on-any-devices
```

## Start

```sh
codex-on-any-devices up
```

The CLI also exposes `codexonanydevices` as a compact alias.
